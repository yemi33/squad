Autonomous Dispatch

Engine discovers work from PRDs, PRs, and manual queue — routes to the right agent based on skills and availability.

Plan Lifecycle

Plan → Review → Approve → PRD → Execute → Verify. Human-in-the-loop at every gate.

Git Worktree Isolation

Each agent works in its own worktree. No conflicts, no cross-contamination, safe parallel execution.

Knowledge Consolidation

Agent findings auto-consolidate into team notes and a searchable knowledge base. The squad learns as it works.

PR Integration

Auto-syncs PRs from Azure DevOps and GitHub. Detects review feedback, build failures, and human comments.

Live Dashboard

Real-time mission control on port 7331. Agent status, work items, PRD progress, dispatch queue, and metrics.

Quick Start

# Install
npm install -g @yemi33/squad

# Initialize — scans for repos, starts engine + dashboard
squad init

# Open dashboard
open http://localhost:7331

# Use the Command Center to create your first plan
# Or add work items directly from the dashboard

Architecture

  │                                                       │
  │   engine.js ──── 60s tick loop ────┐                 │
  │       │                            │                  │
  │   discover work    spawn agents    poll PRs           │
  │       │                │              │               │

  │                                                       │
  │   dashboard.js ── port 7331 ── mission control       │
